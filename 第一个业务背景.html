1.1 数据预处理

我们对于客户画像采用了资产状况，用户粘性和客户持有金融产品风险大小三个维度进行构建。

资产状况维度

我们采用了原始数据中AUM总值这一个特征进行构建。

用户粘性维度

我们采用了三个月六个月转出额度这两个特征进行构建。

风险大小维度

我们采用了十种理财产品的持有标志进行构建，对于字符型变量，我们将其转坏为数字然后再将这些数字转换为浮点数类型。代码如下

data = pd.read_csv('/Users/zhaochenhao/Desktop/data.csv',encoding_errors = 'ignore')#读入数据
data1 = data[data['CUST_AUM']<1000000]#小于100万客户
data2 = data1[['DEP_SA_FLAG','C_FUND_FLAG','D_FUND_FLAG','S_FUND_FLAG','DEP_TD_FLAG','CER_BOND_FLAG','BK_BOND_FLAG','QSQ_FLAG','LOAN_FLAG','CRED_FLAG']]#选出flag
data2.replace('N','0',inplace=True)#替换
data2.replace('Y','1',inplace=True)
data2.replace("B",1,inplace=True)
data2.replace("C",1,inplace=True)
data2 = data2[['DEP_SA_FLAG','C_FUND_FLAG','D_FUND_FLAG','S_FUND_FLAG','DEP_TD_FLAG','CER_BOND_FLAG','BK_BOND_FLAG','QSQ_FLAG','LOAN_FLAG','CRED_FLAG']].astype(float)#改变数据类型

标准化变换

对于三个维度构成的新的三位矩阵，我们采用标准化变化处理这个矩阵方便聚类使用。代码如下：

from sklearn.preprocessing import MinMaxScaler
pro_user = MinMaxScaler().fit_transform(UserPort)
col_names = ['Value','Risk','Attrition']
pro_user = pd.DataFrame(pro_user,columns = col_names)
